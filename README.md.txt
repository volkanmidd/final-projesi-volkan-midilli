# Offline Yapay Zeka Döküman Asistanı
    Bu proje, internet bağlantısı olmadan tamamen yerel donanımda çalışan bir doküman analiz asistanıdır.
    
    ## Nasıl Çalıştırılır?
    1. Ollama'yı indir ve kur (ollama.com).
    2. Modelleri kur: `ollama run qwen2.5:1.5b`, `ollama run phi3`
    3. Terminalde şu komutu yaz: `pip install -r requirements.txt`
    4. Uygulamayı başlat: `streamlit run app.py`
